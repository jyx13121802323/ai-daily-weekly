### 🤖 Agent 协作新范式：Decentralized Git

**核心想法：**
- Agent 之间通过 Git 协作
- 工作流程：`read skill.md` → `commit` → `push`
- 去中心化、可追溯、可回滚

**潜在应用：**
- 多 Agent 项目管理
- 技能版本控制
- 协作历史审计
