# 🤖 AI Daily & Weekly Reports

AI 行业日报和周报自动生成系统

## 📁 项目结构

```
ai-daily-weekly/
├── README.md                 # 本文件
├── templates/                # 报告模板
│   ├── daily-template.md     # 日报模板
│   └── weekly-template.md    # 周报模板
├── scripts/                  # 自动化脚本
│   ├── generate-daily.sh     # 生成日报
│   ├── generate-weekly.sh    # 生成周报
│   └── deploy.sh             # 部署脚本
├── daily-reports/            # 日报归档

│   └── 2026-W10/
```

## 🎯 功能特点

### 日报
- 📰 Hacker News AI/ML 热点
- 📚 arXiv 论文推荐
- 🛠️ 新工具/产品
- 💬 社区讨论
- 📊 数据趋势

### 周报
- 📈 本周概览

- 💰 投资趋势
- 🎓 论文精选 Top 3
- 🔮 下周预测

## ⚡ 快速开始

### 生成日报

```bash
cd /root/.openclaw/workspace/ai-daily-weekly
./scripts/generate-daily.sh
```

### 生成周报

```bash
./scripts/generate-weekly.sh
```

### 查看历史报告

- **日报：** `daily-reports/` 目录
- **周报：** `weekly-reports/` 目录

## 📅 定时任务

### 日报 - 每天 10:00
```cron
0 10 * * * cd /root/.openclaw/workspace/ai-daily-weekly && ./scripts/generate-daily.sh
```

### 周报 - 每周五 17:00
```cron
0 17 * * 5 cd /root/.openclaw/workspace/ai-daily-weekly && ./scripts/generate-weekly.sh
```

## 🔧 配置

编辑 `data/sources.json` 配置数据源：

```json
{
  "daily_sources": {
    "hacker_news": {
      "url": "https://hnrss.org/newest?q=AI",
      "enabled": true
    }
  },
  "schedule": {
    "daily_time": "10:00",
    "weekly_day": "Friday"
  }
}
```
